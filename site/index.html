01 / What it does

A complete path from Typst source to the stage.

Timeline as code

Animate values, coordinate blocks, define cuts, and reuse ordinary Typst definitions across every slide.

Vector-native motion

Morph formulas and SVG geometry in the browser without giving up sharp, resolution-independent output.

Presentation-ready

Preview changes live, navigate animation boundaries, inspect a slide overview, and export one self-contained HTML file.

02 / Workflow

Stay in Typst.
Ship to any browser.

  1. 1
    Describe

    Build slides with familiar Typst markup and Glissade’s small animation API.

  2. 2
    Preview

    Use the live editor to scrub frames and rebuild only the slide you are working on.

  3. 3
    Present

    Export compressed SVG frames and the browser runtime into a single file. Try the example deck.

03 / Origin

Built on a strong foundation.

Glissade grew from Kino by Augustin Albert, whose work established the core variable-animation and SVG-extraction machinery. Glissade carries that foundation into a browser-focused presentation system.
